This project demo how to use http to fetch data from "file" or "rest api"

1. Create project 
ng new FetchUsingHttp

2. Add service "config"
ng g s Config --spec false

3. Copy code from this folders service file 

4. copy girish.json in assets folder

5. Run application 


Here you will see that service can be injected in component class. 
Service will fetch data using http. 
You can create interface to map json object / xml object on given interface object
You can specify interface type while getting data from http and this will save lot of code.
